School Management System - PHP MySQL
A school management system is a comprehensive software solution designed to streamline administrative tasks and facilitate communication between various stakeholders within an educational institution. It typically includes features such as student information management, attendance tracking, gradebook management, scheduling, examination management, financial management, and communication tools for teachers, students, and parents. The system aims to improve efficiency , enhance organization, and promote collaboration among school staff, students, and parents for better educational outcomes.

BBC News Scraper
This project utilizes Node.js with Axios and Cheerio to scrape data from the BBC website. It aims to gather various content categories such as promos, news, sports, and regional updates. The process involves fetching the HTML content of the BBC homepage, parsing it using Cheerio, and then extracting relevant information from specific HTML elements.
